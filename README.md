# Agilify

This is a small compile standup app that based on the commits progress of the working day.

### You can:
  * Add your github project
  * View a list of projects
  * View a list of project commits

### The format of the list of commits the following:
  Yesterday (or on Friday) I commit messsage 1, commit message 2, .. and commit message n in project project 1. Also I commit messsage 1, commit message 2, .. and commit message m in project project 2. â€¦ And I commit messsage 1, commit message 2, .. and commit message k in project project P
